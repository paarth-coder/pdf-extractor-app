# pdf-extractor-app
# PDF to Excel Extractor 🔥

A Python app to extract model-wise serial numbers from PDF invoices and convert them into Excel.

## Features
- Drag & Drop PDF support
- Extracts model-wise serial numbers
- Column-wise Excel output
- Qty vs Extracted Count validation

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run the app
python app.py

## Usage
- Drag and drop PDF into the app  
OR  
- Click "Select PDF"

Excel file will be generated automatically.

## Tech Used
- Python
- pdfplumber
- pandas
- tkinter
